Шаблон проекта с настроенным сборщиком Webpack

```bash
npm install
``` 
-  устанавливает и управляет зависимостями вашего проекта, скачивая пакеты из реестра npm в папку `node_modules` в соответствии с информацией в файлах `package.json` и `package-lock.json`. Без аргументов она устанавливает все зависимости, указанные в файле `package.json`, тогда как при указании имени пакета (`npm install <package-name>`) устанавливает конкретную библиотеку. 

```bash
npm run dev
``` 
- запускает локальный сервер
